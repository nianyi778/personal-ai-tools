# 静かな旅 · 工作流

## 账号人设

**定位：定居日本的中国人，用异乡人的眼睛记录身边的安静。**

- 不是游客（走马观花），不是日本人（习以为常）— 这个视角天然稀缺
- 叙事口吻：第一人称，但克制，不说"我觉得很美"，而是描述细节让读者自己感受
- 正文不解释"为什么来日本"，把生活感自然融入场景描写即可
- 评论区可以真实互动，偶尔提到自己住在哪个城市/季节感受

**个人简介参考（用户自选）：**
```
住在日本 / 不做攻略
把身边没人注意的角落，剪给你看
🎧 建议戴耳机
```

## 账号铁律（每次都必须遵守）

| 规则 | 说明 |
|------|------|
| 标题 / 正文 | 一律**日文** |
| 抖音话题标签 | **最多 5 个** |
| 封面取帧 | 必须从 `merged.mp4`，**绝对不能**用 `_final.mp4`（已烧字幕） |
| 封面文字 | 生成后目视确认所有文字清晰可读；标题区背景过亮则换帧或调暗中央渐变 |
| 字幕样式 | 日文（上）+ 中文（下），PingFang SC 16px，白色描边，无黑底 |
| 视频时长 | 4:30 ～ 5:30 |
| 发布前 | 查 発布記録.md，避免重复发布同一素材 |

## 完整流程（新视频）

### STEP 1 · 查重 + 素材确认
```bash
# 先看发布记录，确认素材未发布过
cat ~/Downloads/静かな旅/静かな旅_発布記録.md
```

### STEP 2 · 下载视频
```bash
yt-dlp -F '<URL>'   # 查看画质
yt-dlp -f "401+140" --merge-output-format mp4 \
  -o "~/Downloads/<地名>.mp4" '<URL>'
```

### STEP 3 · 下载双语字幕
```bash
# 日文（加 --refresh 避免缓存返回错误字幕）
bun ~/.claude/skills/baoyu-youtube-transcript/scripts/main.ts '<URL>' \
  --languages ja --format srt --refresh -o /tmp/<地名>_ja.srt

# 中文
bun ~/.claude/skills/baoyu-youtube-transcript/scripts/main.ts '<URL>' \
  --languages zh-Hans --format srt -o /tmp/<地名>_zh.srt
```

### STEP 4 · 规划分集
- 通读字幕，识别地点 / 主题分区
- **第一条出综合精华**（5min，覆盖所有亮点）
- 再拆 2～4 条主题条，内容不重叠
- 每条 4:30～5:30，选 4～7 个片段，每段 20～70 秒

**开头前3秒铁律（用户平均只看6～12秒）：**
- 第一帧必须是全片最美/最强的画面，直接上，无过渡
- 禁止：黑屏渐入、片头 logo、文字说明、字幕介绍
- 前3秒纯画面 + 环境音，让画面先说话
- 雨天/雾气/水面/夜景开头留存率最高，优先选

### STEP 5 · 剪辑（批量脚本）
复制 `kanazawa_batch.py`，修改顶部三个变量后运行：
```python
VIDEO  = "~/Downloads/<地名>.mp4"
JA_SRT = "/tmp/<地名>_ja.srt"
ZH_SRT = "/tmp/<地名>_zh.srt"
```
脚本自动完成：切片 → concat → 双语字幕对齐 → 烧录 → 输出 `<地名>_<key>_final.mp4`

### STEP 6 · 制作封面

#### 6a · 取帧
```bash
# 从 merged.mp4 取帧（多取几个，选最美的）
for t in 20 60 100 150; do
  ffmpeg -y -ss $t -i "~/Downloads/<地名>_clips/<key>/merged.mp4" \
    -vframes 1 /tmp/frame_${t}.png 2>/dev/null
done
```

#### 6b · 选帧标准（文字可读性优先）
优先选**标题区（画面中央 40-60% 高度）自然偏暗**的帧：

| 好帧 ✓ | 差帧 ✗ |
|--------|--------|
| 瀑布、水面、树荫、夜景 | 天空、电线杆、白墙建筑 |
| 画面中央有阴影或景深虚化 | 中央有高亮反光或杂乱元素 |
| 色调整体偏冷/深 | 大面积浅灰、米白、雪景 |

**如果找不到好帧**，在 cover config 里单独加 `bg_filter` 压暗原图：
```python
"bg_filter": "saturate(0.55) brightness(0.65)",  # 亮帧专用
```

**重要**：`bg_filter` 只作用于 `.bg` 背景层，与文字层完全隔离（文字在 z-index:20，背景在 z-index:0），filter 不会压暗文字。层级结构：
```
body
 ├── .bg               z-index:0  ← filter 在这里，只压背景
 ├── .gradient-overlay z-index:2  ← 上下渐变遮罩
 ├── .title-mask       z-index:5  ← 标题区椭圆暗化
 └── .content          z-index:20 ← 所有文字，不受 filter 影响
```

#### 6c · 生成 & 验证
```bash
# 修改 make_covers.py 顶部 COVERS 列表，填入帧路径和文字
python3 ~/Downloads/静かな旅/make_covers.py
```

**生成后必须目视检查**：日文主标题、英文副标题、三行 mood 文字是否全部清晰可读。若任意文字与背景对比不足，回到 6b 换帧或调参，重新生成。

### STEP 7 · 撰写文案（日文）

生成每条视频的文案后，**立即写入 txt 文件**，路径：
`~/Downloads/静かな旅/<地名>_copy.txt`

文件格式（每条用 `---` 分隔）：
```
=== 01_highlight ===
タイトル：xxx

场景描写句1（画面 + 感官词，如：「湯煙の向こうに、赤い橋が浮かぶ。」）
场景描写句2（氛围/细节，如：「硫黄の香りと、川の音だけが耳に届く。」）
情感收尾句（留白/悬念，不要总结，如：「ただ、歩いた。」）

🎧 イヤホン推奨
#天ヶ瀬温泉 #大分旅行 #日本秘湯 #癒し系 #没入旅行

---
=== 02_xxx ===
...
```

**文案风格规范：**
- 句子短，留白多，动词结尾（「歩いた」「流れる」「消えた」）
- 禁用总结句（「素晴らしい旅でした」类）— 让画面说话
- 场景描写优先感官词：声音、气味、温度、光线

**标题句式（实测有效）：**

| 类型 | 结构 | 示例 |
|------|------|------|
| 情感转折 ✓ | ～だから、～になった | 「雨の日だから、この町が好きになった」|
| 限定揭秘 ✓ | ～にしか見せない～がある | 「雪の日にしか見せない顔がある」|
| 地名 + 悬念 | ～、～に隠れている | 「天ヶ瀬、滝の音の中に隠れている」|

**避开：** 中文标题（播放量高但主页访问=0，涨粉无效）

**话题标签 5 个分配规则：**

| 位置 | 类型 | 示例 |
|------|------|------|
| 1 | 地名精准标签 | #天ヶ瀬温泉 |
| 2 | 地区/县标签 | #大分旅行 |
| 3 | 内容类型标签 | #日本秘湯 / #まち歩き / #温泉街散策 |
| 4 | 情绪/风格标签 | #癒し系 / #ひとり旅 / #没入旅行 |
| 5 | 流量大标签 | #日本旅行（固定） |

写完后告知用户文件路径，方便直接打开复制。

### STEP 8 · 发布节奏（一组多条视频）

同一地点的 4 条视频**按以下间隔发布**，不要一次全发：

```
Day 1  → 01_highlight（精华）   ← 算法推荐期，覆盖最广受众
Day 4  → 02_主题条              ← 留住精准用户
Day 8  → 03_主题条              ← 深度用户
Day 12 → 04_主题条              ← 忠实粉丝
```

发布时间：**晚上 8～10 点**（抖音黄金时段）

### STEP 9 · 收尾 checklist
- [ ] 视频时长 4:30 ～ 5:30 ✓

- [ ] 标题日文，含地名关键词 ✓
- [ ] 话题标签恰好 5 个，结构符合规则 ✓

## 数据复盘（发布后3天看）

**核心指标优先级：主页访问率 > 完播率 > 2s跳出率 > 播放量**

播放量高≠账号增长，主页访问才是真实兴趣的体现。

| 指标 | 健康值 | 问题信号 |
|------|--------|---------|
| 2s跳出率 | <35% | >45% 说明开头钩子太弱 |
| 主页访问率 | >1.5% | <0.5% 说明内容没触动受众 |
| 平均播放时长 | >20s | <10s 说明开头失效 |
| 完播率 | >1.5% | 对5min视频参考价值有限 |

**已知规律（来自实测数据）：**
- 中文标题：播放量高，主页访问=0，涨粉无效 → 禁用
- 日文情感句式标题：主页访问率3～6%，是最有效的转化格式
- 雨天/冬季/雾气素材：平均留存时长比普通素材高2倍

## 选题优先级

**优先选（差异化强）：**
- 小众温泉、山间小镇、宿场町、城下町
- 雨天、冬季、晨雾、夜景 — 情绪稀缺内容
- 「誰もいない」场景 — 账号核心差异化标签
- 开湯历史悠久的温泉（1000年以上优先）

**避开：**
- 东京、大阪、京都热门景点 — 竞争过激，账号定位不符
- 美食探店、购物 — 偏离沉浸漫步定位
- 夏季人流密集场景 — 破坏孤独感
